# all-agents

Briding between Cursor and Claude code AI configuration, making it work as much as possible for web / cloud agents too that don't have the full capabilities of their local counterparts.

## Functionality / Prompts

### Commands

- `how-to-prompt` -> @docs/meta/PROMPTING.md
- `optimize-prompt` -> @docs/meta/OPTIMIZE-PROMPT.md
- `meta/claude-code/create-command` -> Web docs, @docs/meta/PROMPTING.md, @docs/meta/CLAUDE-CODE-TOOLS-PERMISSIONS.md
- `meta/claude-code/create-agent` -> Web docs, @docs/meta/PROMPTING.md, @docs/meta/CLAUDE-CODE-TOOLS-PERMISSIONS.md
- `meta/claude-code/create-plugin` -> Using create-plugin.ts script and hardcoded info
- `meta/claude-code/create-skill` -> Using create-skill.py script and hardcoded info
- `meta/create-cursor-rule` -> Hard coded docs, @docs/meta/PROMPTING.md
